# Wallet System - Quick Start Guide

## ğŸš€ Get Started in 3 Minutes

### Step 1: Navigate to Wallet Page

**For Clients:**

```
1. Login as client
2. Click "My Wallet" in sidebar (2nd menu item)
3. You're now at /client/wallet
```

**For Workers:**

```
1. Login as worker
2. Click "My Wallet" in sidebar (2nd menu item)
3. You're now at /worker/wallet
```

### Step 2: View Your Balance

On the wallet page, you'll immediately see:

- Your available balance (in USD)
- Pending transactions
- Total earned/spent
- Active escrows
- Two action buttons: Deposit & Withdraw

### Step 3: Add Funds (Deposit)

**Option A: Bank Transfer (Vietnamese Banks)**

```
1. Click "Deposit" button
2. Select "Bank Transfer" tab
3. Enter amount (e.g., 100 USD)
4. Click "Generate QR Code"
5. Scan QR with your banking app
6. Transfer with exact content shown
7. Wait 1-5 minutes for confirmation
8. Balance updates automatically âœ…
```

**Option B: PayPal**

```
1. Click "Deposit" button
2. Select "PayPal" tab
3. Enter amount (e.g., 50 USD)
4. Click "Pay with PayPal"
5. Complete payment on PayPal site
6. Redirected back to platform
7. Balance updates immediately âœ…
```

### Step 4: Withdraw Funds

**Option A: To Bank Account**

```
1. Click "Withdraw" button
2. Select "Bank Transfer" tab
3. Enter amount (minimum $50)
4. Enter bank name (e.g., Vietcombank)
5. Enter account number
6. Enter account holder name
7. Click "Submit Withdrawal Request"
8. Wait 1-3 business days
9. Receive funds in bank account âœ…
```

**Option B: To PayPal**

```
1. Click "Withdraw" button
2. Select "PayPal" tab
3. Enter amount (minimum $50)
4. Enter PayPal email
5. Click "Submit Withdrawal Request"
6. Wait 1-2 business days
7. Receive funds in PayPal âœ…
```

### Step 5: Track Transactions

Scroll down on the wallet page to see:

- Complete transaction history table
- Filter by type (deposit, withdrawal, etc.)
- Filter by status (pending, completed, etc.)
- Filter by date range
- Sort by any column
- Pagination for many transactions
